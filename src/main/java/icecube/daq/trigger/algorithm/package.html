Provides classes that implement trigger algorithms.

AbstractTrigger implements the ITriggerConfig, ITriggerControl, and ITriggerMonitor interfaces. It implements all of the methods of these interfaces except for:

  • runTrigger(): void
  • flush(): void
  • isConfigured(): boolean

In addition, the following methods must be overridden for trigger algorithms that take parameters or readout requests:

  • addParameter(TriggerParameter): void
  • addReadout(TriggerReadout): void

Each trigger algorithm extends AbstractTrigger.
